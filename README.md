# News AI Dashboard

A full-stack AI-powered news dashboard built with React, Express, MongoDB, NewsAPI, and Gemini.

## Features

- Live news by category
- News search
- Article details
- AI-generated 3-point summaries
- User registration and login
- JWT authentication
- Save and delete summaries
- User profile with profile image
- Loading and error handling
- Session storage caching

## Tech Stack

### Frontend
- React
- Vite
- CSS
- NewsAPI
- Gemini API

### Backend
- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- bcrypt
